# Plantillas

## Plantillas documentos UGR

El objetivo del repositorio [Plantillas-UGR](https://github.com/latex-mat-ugr/Plantillas-UGR) es la creación de unas clases de LaTeX para la creación de documentos (certificados, cartas, exámenes,...) que respeten el manual de identidad visual corporativa de la Universidad de Granada. Para ello se ha hecho uso tanto de los elementos de identidad visual corporativa como del repositorio de documentos administrativos de la Universidad de Granada. 

## Plantilla TFG

En el [repositorio Plantilla-TFG](https://github.com/latex-mat-ugr/Plantilla-TFG) se ha desarrollado una plantilla para la realización de un TFG mediante LaTeX que cumple con los requisitos para la memoria de TFG del Grado de Matemáticas de la Universidad de Granada.

Anteriormente, Mario Román [@M42](https://github.com/M42) y David Charte [@fdavidcl](https://github.com/fdavidcl) habían elaborado unas plantillas de TFG y beamer para el Doble Grado de Matemáticas e Informática basadas en [classicthesis](https://ctan.org/pkg/classicthesis) y [metropolis](https://ctan.org/pkg/beamertheme-metropolis). Las plantillas están disponibles en este [repositorio](https://github.com/M42/templates) de GitHub.

## Plantilla TFM

Además en esta carpeta hay un ejemplo de una portada de TFM del Máster de Matemáticas. Esta portada se puede utilizar también para un TFG, cambiando el fondo por el que propone Mario [ugrA4.pdf](https://github.com/M42/templates/blob/master/classicthesis-tex-es/ugrA4.pdf).

Las imágenes de fondo están extraídas de la página de la Universidad de Granada.
